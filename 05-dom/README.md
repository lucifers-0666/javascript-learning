# 🌐 DOM (Document Object Model)

This folder contains my DOM learning notes and code examples.

## Topics Covered

| File | Topic |
|------|-------|
| `01-selecting-elements.js` | querySelector, getElementById, etc. |
| `02-manipulating-elements.js` | innerHTML, textContent, classList |
| `03-events.js` | addEventListener, event types, event object |
| `04-event-bubbling-delegation.js` | Bubbling, capturing, delegation |
| `05-creating-removing-elements.js` | createElement, append, remove |
| `06-dom-traversal.js` | parentNode, children, siblings |
| `07-forms.js` | Form events, input handling, validation |
| `08-styles-and-attributes.js` | Inline styles, getAttribute, setAttribute |

## How to Run

Open `index.html` in a browser and open DevTools Console to see output.
